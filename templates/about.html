Smart-Load

О проекте

Smart-Load — это веб-приложение для управления учебными задачами, разработанное в рамках школьного проекта по информатике.

Основная идея проекта — помочь школьникам эффективно планировать своё время, не забывать о домашних заданиях и снизить учебный стресс за счёт наглядного списка дел и системы напоминаний.

О проекте

Проект создан в образовательных целях для демонстрации навыков веб-разработки и изучения следующих тем:

  • Создание веб-приложений на фреймворке Flask
  • Проектирование и работа с базами данных (SQLAlchemy, PostgreSQL)
  • Реализация аутентификации и авторизации пользователей
  • Разработка клиентской части на HTML, CSS и JavaScript
  • Создание REST API для взаимодействия с сервером
  • Система автоматических напоминаний о дедлайнах

Данный проект является выпускной работой и демонстрирует полученные знания в области информатики и веб-технологий.

Основные возможности

Управление задачами

Создание, выполнение и удаление учебных задач с указанием сроков

Умные напоминания

Автоматические уведомления о приближающихся дедлайнах

Личный кабинет

Регистрация и авторизация — каждый пользователь видит только свои задачи

Адаптивный дизайн

Удобная работа с компьютера, планшета или телефона

Технологии

Python 3.11 Flask SQLAlchemy PostgreSQL HTML5/CSS3 JavaScript ES6 Flask-Login

Актуальность проекта

Современные школьники сталкиваются с высокой учебной нагрузкой. Отсутствие системы планирования приводит к забытым дедлайнам, спешке и стрессу. Существующие приложения либо перегружены функциями, либо не учитывают специфику учебного процесса. Smart-Load решает эту проблему, предоставляя простой и понятный инструмент для управления учебными задачами.

Планы развития

  • Добавление категорий задач по предметам
  • Визуальный календарь для распределения задач по дням
  • Экспорт задач в PDF для печати
  • Мобильное приложение на React Native
  • Интеграция с электронным дневником
  • Создание аккаунтов для учителей и родителей — чтобы родители могли контролировать планы учеников, а учителя понимали, ждать ли ученика на уроке, выполнил ли он домашнее задание или нет
